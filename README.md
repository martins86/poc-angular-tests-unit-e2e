# Comandos NPM
```
▶ ng add @angular/material   /// Package angular material.
```

```
install: npm install standard husky@next --save-dev

▶ npm i standard -D        /// Define e aplica o padrão de código.
▶ npm i husky@next -D      /// Executa scripts antes do commit.
```
